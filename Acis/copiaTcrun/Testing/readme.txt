Instalación
    ~$ npm install --dev

Para correr las pruebas utilizar el comando:
    ~$ npm test

El código de los tests se encuentran en el directorio test.
La convención de nombres será: "<modelo en plural>Test.js", ejemplo "projectsTest.js"

Tomen como ejemplo el archivo de pruebas todoTest.js - Este archivo utiliza la API del tutorial de Dan.
Como observarán cuenta con una pequeña descripción del archivo al inicio.
Las pruebas se agrupan en distintas categorias que cuentan con una descripción.
Cada método de prueba también cuentan con una descripción (caso, qué se espera).

El nombre del test es representativo de lo que hace o prueba (parametros nulos, petición válida ...).
